Asignar tamaño de una etiqueta definida

Estas clases, heredan los valores del tamaño de fuente de una etiqueta determinada, por ejemplo, si usamos font_size_h1 en un elemento, este heredará el tamaño que se le asignó a la etiqueta h1. Esto implica que si cambiamos en algún momento el tamaño del h1 también cambiará el tamaño del elemento con la clase, en cambio con la clase .font_size_12, que nos da el mismo resultado en tamaño, los textos con esta clase mantendrán su tamaño si en algún momento decidimos que las etiquetas h1 tengan más o menos tamaño.

Asignar estilo de una etiqueta definida

Tal como con las clases anteriores pudimos heredar el tamaño de fuente de las etiquetas asociadas, podemos hacerlo también con todo su estilo, tamaño, fuente, color por defecto, interlineado, etc.
Con esto, por ejemplo, podemos hacer que una etiqueta p se vea como un h1 con el fin de controlar el SEO de nuestra página para no tener más de una etiqueta H1 y colaborar con la indexación.

.font_style_h1

Asigna al elemento todos los estilos definidos para la etiqueta h1, como tamaño, color, alto de línea, etc.
