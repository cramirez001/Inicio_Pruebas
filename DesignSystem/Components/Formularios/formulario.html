Formularios

Para construir formularios usaremos algunas clases exclusivas.

Primero, vamos a declarar que es un formulario el que estamos construyendo. Esto con la clase .cpl_flex_form.

<div class="cpl_flex_form"> .. </div>

Luego tendremos que definir la distribución con el sistema de grillas, crearemos un row .flex_row y dentro de este los contenedores de los campos a modo de columna .cpl_col, digamos que queremos un formulario de dos columnas, iremos agregando estos contenedores de campo con .cpl_col .cpl_col_6. A estos contenedores además le añadiremos la clase .input_wrap que nos ayudará a controlar elementos asociados al campo como los mensajes de error. <div class="cpl_flex_form"> <div class="flex_row"> <div class="cpl_col cpl_col_6 input_wrap"> ... </div> <div class="cpl_col cpl_col_6 input_wrap"> ... </div> </div> </div>

Ahora, agregaremos los campos, en este ejemplo agregaremos dos campos de texto. Para que tenga los estilos definidos para los campos debemos agregar a este input la clase .cpl_input <div class="cpl_flex_form"> <div class="flex_row"> <div class="cpl_col cpl_col_6 input_wrap"> <input class="cpl_input" type="text" /> </div> <div class="cpl_col cpl_col_6 input_wrap"> <input class="cpl_input" type="text" /> </div> </div> </div>

Finalmente agregamos los labels con un span y tendremos algo así: <div class="cpl_flex_form"> <div class="flex_row"> <div class="cpl_col cpl_col_6 input_wrap"> <label class="label_l"> <span>Campo de texto</span> </label> <input class="cpl_input" type="text" /> </div> <div class="cpl_col cpl_col_6 input_wrap"> <label class="label_l"> <span>Campo de texto</span> </label> <input class="cpl_input" type="text" /> </div> </div> </div>
